Overview

This project is an interactive Rock, Paper, Scissors game that allows users to play against the computer. The goal is to provide a fun and simple game experience that mimics the classic hand game.

Technologies Used

The project was built using vanilla JavaScript for game logic and DOM manipulation. CSS was applied to style the game elements, and HTML was used to create the basic structure of the buttons and display areas.

Challenges

One of the main challenges was ensuring the randomness of the computer's choice and providing feedback to the player immediately after each round. Another challenge was keeping track of scores and resetting the game when required.

Outcome

The final product is a simple, fast, and engaging Rock, Paper, Scissors game that is responsive and easy to play. The game works smoothly on all modern browsers and devices, offering an enjoyable experience for players of all ages.
