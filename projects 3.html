Overview

This project is a Positive Affirmation Generator that provides users with a random positive affirmation each time they click a button. It aims to inspire and uplift users with motivational messages, promoting a positive mindset.

Technologies Used

The generator was developed using HTML, CSS, and JavaScript. It features a simple and intuitive user interface, allowing users to easily generate new affirmations with a single click.

Challenges

One of the challenges was sourcing a diverse set of positive affirmations. I also needed to ensure that the JavaScript function effectively randomized the affirmations and displayed them correctly in the UI.

Outcome

The Positive Affirmation Generator is fully functional and has received positive feedback for its simplicity and effectiveness. Users can refresh their affirmations with each click, contributing to a more uplifting daily routine.
